# CRUD系统

#### 介绍
一个基于SSM简单的增删改查系统

#### 软件架构
前后端不分离项目
利用jquery,jsp进行前后端交互
spring + springMVC + Mybatis 三件套
数据库用的是Mysql 服务器用的是tomcat
